# 주식거래대금순위

Notion 거래대금DB를 Cloudflare Worker 프록시로 연동하는 단일 파일 HTML 분석 앱입니다.

**URL**: https://shinfund.github.io/stock-volume-rank/

---

## 앱 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 앱 목록 랜딩 페이지 |
| `stock-volume-rank.html` | 주식거래대금순위 분석 앱 |

---

## 탭 구성

| 탭 | 설명 |
|---|---|
| 일별 상위 | 선택 날짜의 거래대금 상위 종목 및 순위 테이블 |
| 순위 변동 | 전일 대비 순위 상승·하락·신규 진입 현황 |
| 상위 진입 | 기준 순위(상위 5·10·20·30위) 신규 진입 종목 |
| 순위 분포 | 날짜별 순위 밴드(1위·2~3위·4~5위 등) 분포 |
| 종목 이력 | 특정 종목의 기간 내 순위 추이 및 일별 상세 |

## 변경 이력

### 2026-06-14

#### 최초 구현
- Notion 거래대금DB 실시간 연동 (Cloudflare Worker 프록시)
- 5개 탭 구성: 일별 상위·순위 변동·상위 진입·순위 분포·종목 이력
- 기간 드롭다운 (1·5·10·20·60·120일·전체)
- 데모 데이터 폴백 (Worker 미연결 시 자동 표시)
- localStorage 설정 저장 (Worker URL, DB ID)

#### 버그 수정
- **이중 `+` 연산자 NaN 버그** — `.map()` arrow concise body에서 `+'<td>'`가 unary `+`로 파싱되어 테이블 셀이 NaN으로 깨지는 문제 수정 (renderEntry·renderDaily·renderHistory 3곳)
- **null/NaN 안전 처리** — `numOr()` 함수에 `!isFinite()` 체크 추가, `safeN()` 헬퍼 도입으로 포맷 함수 전반 NaN 방어 강화
- **`setSelDate()` 오작동** — `renderDaily()`만 호출하던 문제를 `renderCurrent()`로 수정, 상위진입 탭 "보기" 버튼이 정확한 탭을 재렌더하도록 수정

#### 기능 추가
- **년/월/일 3단계 날짜 드롭다운** — 일별 상위·순위 변동·상위 진입·순위 분포 4개 탭에 적용
- **순위 분포 탭 날짜 필터** — 기간 누적 집계에서 선택 날짜 하루 분포로 변경

#### 코드 개선
- **XSS 방어** — onclick 속성 내 동적 날짜 값에 `esc()` 적용
- **`chgCls()` 호출 통일** — `r.chg||0` 혼용 제거, `r.chg` 단일 방식으로 통일
- **데드코드 제거** — `init()` 내 미도달 `else` 브랜치 제거
- **섹션 타이틀 정리** — 순위변동 탭 제목의 불필요한 "vs" 비교 날짜 텍스트 제거

#### 반응형 최적화
- 4단계 브레이크포인트 적용
  - `≤480px` 소형 모바일: 로고 서브텍스트 숨김, 폰트·패딩 최소화
  - `≤600px` 모바일: 종목코드 숨김, 터치 타겟 min-height 40px
  - `≥769px` PC: KPI 카드·테이블 폰트 확대, 패딩 여유 증가
  - `≥1100px` 와이드 PC: KPI 숫자 20px, 순위분포 3열 그리드

#### 인덱스 페이지
